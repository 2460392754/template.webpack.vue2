## dev - 开发环境

```
$ yarn dev
```

## build - 构建环境

```
$ yarn build
```

## build:analyzer - 在构建环境中进行文件分析

```
$ yarn build:analyzer
```

## build:dll - 把第三方公共模块构建出来, 加快构建速度

```
$ yarn build:dll
```

## format:fix - 代码格式化

```
$ yarn format:fix
```

## lint:fix - 代码校验并修复

```
$ yarn lint:fix
```
