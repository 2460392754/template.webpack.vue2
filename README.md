## dev - 开发环境

```
$ yarn dev
```

## build - 构建环境

```
$ yarn build
```

## build:analyzer - 在构建环境中进行文件分析

```
$ yarn build:analyzer
```

## format:fix - 代码格式化

```
$ yarn format:fix
```

## lint:fix - 校验代码 js 并修复

```
$ yarn lint:fix
```

## lint:fix - 校验代码 css 并修复

```
$ yarn lincsst:fix
```
